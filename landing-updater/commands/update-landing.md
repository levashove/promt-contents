# /landing-updater:update-landing

Запускает агента для обновления текста лендинга с сохранением структуры и SEO-оптимизацией.

## Usage

```
/landing-updater:update-landing [url]
```

Или просто напишите:
- "Обнови лендинг https://..."
- "Актуализируй текст страницы"
- "Нужно обновить контент лендинга"

## Workflow

```
┌─────────────────────────────────────────────────────────┐
│  1. ПОЛУЧЕНИЕ ТЕКСТА                                   │
│     URL страницы или текст напрямую                     │
├─────────────────────────────────────────────────────────┤
│  2. РАЗБОР НА БЛОКИ                                    │
│     Автоматическая сегментация                          │
├─────────────────────────────────────────────────────────┤
│  3. SEO-ПАРАМЕТРЫ                                      │
│     Ваши ключи или наши предложения                     │
├─────────────────────────────────────────────────────────┤
│  4. РЕФЕРЕНСЫ И УКАЗАНИЯ                               │
│     Страница-образец или текстовые инструкции           │
├─────────────────────────────────────────────────────────┤
│  5. АНАЛИЗ И ПРЕДЛОЖЕНИЯ                               │
│     Что улучшить в каждом блоке                         │
├─────────────────────────────────────────────────────────┤
│  6. ГЕНЕРАЦИЯ                                          │
│     Обновлённый текст с сохранением структуры           │
└─────────────────────────────────────────────────────────┘
```

## Input Options

### 1. По ссылке
```
/landing-updater:update-landing https://vkcloud.kz/containers/
```

### 2. Текст напрямую
```
Обнови этот текст лендинга:

# VK Cloud
Облачная платформа для цифровизации бизнеса...
```

### 3. Файл
Загрузите документ с текстом лендинга (TXT, MD, DOCX).

## SEO Parameters

Агент запросит или предложит:

| Параметр | Описание | Пример |
|----------|----------|--------|
| Основные ключи | Главные ключевые слова | "облачная платформа, VK Cloud" |
| LSI-ключи | Дополнительные слова | "виртуальные серверы, Kubernetes" |
| Title | Meta-заголовок | "VK Cloud — облачная платформа для бизнеса" |
| Description | Meta-описание | "Облачные сервисы VK Cloud: серверы, K8s, БД..." |

## Reference Options

1. **URL референса** — страница с желаемым стилем
2. **Текстовые указания** — что изменить, на что обратить внимание
3. **Оба варианта** — и референс, и инструкции

## Output

### Формат "Было/Стало"

```markdown
══════════════════════════════════════════════════════════
БЛОК 1: HERO
══════════════════════════════════════════════════════════

**Было:**
# VK Cloud
Облачная платформа для цифровизации бизнеса

**Стало:**
# VK Cloud
Облачные сервисы для разработки и масштабирования продуктов

📝 *Изменения: добавлен ключ, конкретизирована ценность*
```

### Чистый текст
Отдельный блок с готовым текстом для копирования.

### Статистика
```
| Метрика | Было | Стало |
|---------|------|-------|
| Слов | 450 | 460 |
| Ключевых слов | 3 | 8 |
| Цифр/фактов | 2 | 5 |
```

## Commands

### Навигация

| Команда | Действие |
|---------|----------|
| `назад` | Предыдущий этап |
| `статус` | Текущее состояние |
| `блок N` | Редактировать блок N |

### Доработка

| Команда | Действие |
|---------|----------|
| `короче блок N` | Сократить блок |
| `подробнее блок N` | Расширить блок |
| `больше SEO` | Добавить ключей |
| `вернуть блок N` | Откатить к исходному |

## Key Principles

```yaml
preserve:
  ✓ Количество блоков

  ✓ Структура H1 → H2 → текст
  ✓ Общий смысл и позиционирование

update:
  ✓ Формулировки и слова
  ✓ Актуальность данных
  ✓ SEO-оптимизация
  ✓ Устаревшие термины
```
