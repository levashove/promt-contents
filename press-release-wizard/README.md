# Press Release Wizard

Интерактивный агент для написания пресс-релизов в стиле VK Cloud.

## Что делает

Принимает исходные материалы (файл, текст или описание), задаёт уточняющие вопросы и генерирует готовый пресс-релиз профессионального качества.

## Быстрый старт

### Запуск

```bash
/write-press-release
```

Или просто:
```
Напиши пресс-релиз о [продукте]
```

## Workflow

```
┌─────────────────────────────────────────┐
│  1. МАТЕРИАЛЫ                          │
│     Загрузите или опишите продукт       │
├─────────────────────────────────────────┤
│  2. АНАЛИЗ                             │
│     Агент выделяет ключевые факты       │
├─────────────────────────────────────────┤
│  3. УТОЧНЕНИЕ                          │
│     Вопросы для заполнения пробелов     │

│  4. ГЕНЕРАЦИЯ                          │
│     Создание текста                     │
├─────────────────────────────────────────┤
│  5. ДОРАБОТКА                          │
│     Корректировка по замечаниям         │
└─────────────────────────────────────────┘
```

## Быстрые шаблоны

Укажите тип релиза для ускоренного процесса:

| Тип | Описание |
|-----|----------|
| `запуск продукта` | Новый сервис/платформа |
| `партнёрство` | Совместный запуск с партнёром |
| `линейка` | Несколько продуктов |
| `обновление` | Новая версия существующего продукта |

## Пример диалога

```
Вы: Нужен релиз о новом сервисе Cloud Audit

Агент: Расскажите подробнее о Cloud Audit.
       Что это за сервис и какую задачу решает?

Вы: Сервис для проверки безопасности кода.
    Сканирует на уязвимости, для DevSecOps.

Агент: Вопрос 1: Какую проблему решает для клиентов?

Вы: Находит уязвимости до продакшена

Агент: Вопрос 2: Есть конкретные цифры?

Вы: 10 000 сканирований в день, 1 млрд строк кода

Агент: Вопрос 3: Кто даст комментарий?

Вы: Дмитрий Лазаренко, директор по продукту VK Cloud

Агент: Пресс-релиз готов!

       # VK Cloud запустила сервис для повышения безопасности приложений

       VK Cloud представила новый сервис — Cloud Audit...
```

## Типы входных данных

| Тип | Описание |
|-----|----------|
| **Файл** | PDF, DOCX, PPTX, TXT, MD |
| **Текст** | Описание продукта, ТЗ, черновик |
| **Описание** | Словесное описание в чате |

## Уточняющие вопросы

Агент спрашивает только если информации нет в материалах:

- Главная ценность для клиентов
- Целевая аудитория
- Цифры и метрики
- Спикер для цитаты
- Партнёр (если есть)
- Сертификаты безопасности

## Команды доработки

После генерации можно корректировать:

| Команда | Действие |
|---------|----------|
| `короче` | Сократить на 20% |
| `подробнее` | Добавить деталей |
| `другой заголовок` | 3 варианта |
| `переписать цитату` | Новая версия |
| `добавить цифры` | Усилить фактами |

## Экспорт

| Команда | Формат |
|---------|--------|
| `экспорт html` | HTML с SEO-метатегами |
| `экспорт json` | Структурированные данные |
| `экспорт txt` | Plain text для email |
| `формат даты en` | Английский формат даты |

## SEO-элементы

При экспорте в HTML автоматически генерируются:
- `<title>` и `<meta description>`
- Open Graph теги
- Ключевые слова

## Валидация

Агент автоматически проверяет:
- Минимум 2 цифры в тексте
- Минимум 1 цифра в цитате
- Отсутствие запрещённых слов ("инновационный", "значительно")

## Структура релиза

```markdown
{дата} • Пресс-релизы

# {Заголовок}

{Лид-абзац: Что + Зачем + Контекст}

{Описание продукта}

*«{Цитата}»*, — комментирует **{Спикер}**.

{Дополнительные детали}

**{Компания}** — {бойлерплейт}
```

## Стиль VK Cloud

Агент обучен на реальных пресс-релизах VK Cloud:

**DO**
- Конкретные цифры: "выросла на 62%"
- Активный залог: "VK Cloud запустила"
- Короткие предложения

**DON'T**
- Превосходные степени: "инновационный"
- Размытые формулировки: "значительно"
- Канцеляризмы: "осуществляет"
