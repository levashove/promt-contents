# VK Cloud запустила сервис для повышения безопасности приложений

VK Cloud представила новый сервис — Cloud Audit. Он позволяет находить уязвимости в коде до того, как приложение попадёт в продакшн. Сервис интегрируется в CI/CD-пайплайн и автоматически проверяет каждый коммит.

Cloud Audit проводит статический анализ кода, проверяет зависимости на известные уязвимости и анализирует конфигурации инфраструктуры. Сервис поддерживает более 20 языков программирования, включая Python, Go, Java и JavaScript. Результаты сканирования доступны в личном кабинете VK Cloud и интегрируются с популярными системами управления задачами.

Для DevSecOps-команд Cloud Audit предлагает расширенные возможности: настраиваемые политики безопасности, автоматическую приоритизацию уязвимостей по критичности и отчёты для compliance-аудитов.

*«Безопасность становится неотъемлемой частью разработки, а не отдельным этапом в конце. DevSecOps-подход требует инструментов, которые работают со скоростью разработки. Cloud Audit проводит более 10 000 сканирований в день и охватывает более 1 млрд строк кода в инфраструктуре VK. Теперь этот инструмент доступен всем клиентам VK Cloud»*, — комментирует **Дмитрий Лазаренко, директор по продукту VK Cloud**.

Сервис соответствует требованиям 152-ФЗ и включён в реестр российского ПО. Cloud Audit доступен в публичном облаке VK Cloud, а также в формате Private Cloud для компаний с повышенными требованиями к безопасности.

**VK Cloud** — платформа с широким набором облачных сервисов для эффективной разработки и работы с данными для компаний любого масштаба. VK Cloud входит в портфель решений VK Tech и базируется на многолетнем опыте развития интернет-сервисов и технологий на базе открытого кода.
